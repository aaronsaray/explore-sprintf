# explore-sprintf

Explore sprintf() in the browser

## Development

### Coding 

```sh
npm run dev
```

### Run Unit Tests with [Vitest](https://vitest.dev/)

```sh
npm run test
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```

## Deploy

```sh
npm run build
```

Deploy the `/dist` folder.

GitHub actions are deploying to GitHub pages.
